# CMAS Website

Official website draft for the China Multi-Agent Systems Conference (CMAS).

## Overview

This repository contains a bilingual static website for CMAS.

- Chinese and English share one page structure
- the homepage works as a portal page
- each major section has its own standalone page
- the site is ready for GitHub Pages deployment

## Structure

- `index.html`: homepage portal
- `about.html`: conference overview
- `topics.html`: focus topics
- `history.html`: previous editions
- `organization.html`: organizers and chair placeholders
- `attend.html`: attendance information
- `faq.html`: frequently asked questions
- `contact.html`: contact placeholders
- `assets/styles/site.css`: shared styling
- `assets/scripts/site.js`: shared bilingual behavior

## Local Preview

Open `index.html` directly in a browser.

- default language: Chinese
- English version: add `?lang=en` to the page URL
